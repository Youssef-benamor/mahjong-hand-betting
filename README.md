## 💎 The Experience Mahjong Royale strips away the clutter and focuses on a
premium "Casino-Zen" aesthetic. The UI is built using a **12-column responsive
grid** and **Glassmorphism** panels to provide a professional, standalone app
feel. - **Dynamic HUD:** Live score, combo tracking, and deck integrity bar. -
**Fluid Motion:** Tiles utilize `AnimatePresence` for seamless entry and exit. -
**Responsive Core:** Fully playable on desktop, tablet, and mobile.

<br />
